Code scraps

Here are some code scraps in Stata and R that I frequently use.

Stata

Descriptives table: Show summary stats for your sample.

Balance table: Show means and standard deviations with different samples across columns.

Postfile for a bootstrap: Run analyses in a loop and save results in a tempfile.

Coefplot by hand: Save coefficients and standard errors from models and display them in a scatter plot.

Self-documenting tables and figures: Automatically add a footer to .tex tables and figures saying where and when it was made.

Shaded plot: Shade a certain range of x-axis values.

R

Regression table: Make a regression table where outcome means are reported in the table, N changes across columns, and columns have different standard error adjustments (e.g., clusters). The table it makes is here.

Make a pretty descriptives table: Show summary stats for your sample.

Latex

Syntax highlighting: Example code using the minted package. It makes this PDF. Useful for teaching materials. (Note: this works right away in Overleaf, but local users need to have installed Pygments. More here.)
